# LIGNES — Application de gestion de check-lists (React + Vite + TypeScript)

## Démarrer dans StackBlitz (le plus simple)
1. Ouvrir https://stackblitz.com/ → **Upload Project** (ou New Project > Import) et glisser le `.zip`.
2. StackBlitz va installer automatiquement. Cliquez sur **Run** si besoin.
3. L'appli s'ouvre dans le panneau de droite.

## Démarrer en local
```bash
npm install
npm run dev
```
Ouvrir http://localhost:5173

## Fonctionnalités
- Projets / Étapes / Tâches (CRUD, persistance locale)
- Collaborateurs (création + attribution)
- Modèles (2 modèles prédéfinis, applicables à la création d'un projet)
- Export PDF (synthèse tableau de bord et export projet)

## Logo
Le logo LIGNES est utilisé en en-tête (`public/logo.jpg`) et repris dans le PDF.
